# Breakout Day 02

🎮 Segundo projeto do desafio:
criar 1 jogo por dia para estudar desenvolvimento de games e TypeScript.

Já desenvolvo jogos utilizando Unity e Unreal Engine, mas queria aprofundar meus estudos em TypeScript e desenvolvimento web. Então decidi usar pequenos jogos como forma prática de aprendizado.

## Objetivos do projeto

- Estudar TypeScript na prática
- Melhorar arquitetura de gameplay
- Aprender gerenciamento de estados
- Trabalhar lógica de colisão
- Explorar sistemas de progressão
- Publicar protótipos jogáveis online

## Tecnologias

- Phaser 3
- TypeScript
- Vite
- GitHub Pages

## Mecânicas estudadas

- Game Loop
- Input
- Colisão
- Física básica
- Sistema de níveis
- Sistema de vidas
- Destruição de blocos
- Partículas simples
- Pause system
- State Machine

## Estados do jogo

- Playing
- Paused
- LevelTransition
- GameOver
- Win

A / D
